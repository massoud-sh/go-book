| تایپ  |   مقدار پیش فرض یا مقدار صفر           | 
|-----------|--------|
| int   | 0 |
| int8     | 0 |
| int16     | 0 |
| int32     |  0 |
| int64     |  0 |
| uint     | 0 |
| uint8     |  0 |
| uint16     |  0 |
| uint32     |  0 |
| uint64     |  0 |
| uintptr     | 0 |
| float32     |  0 |
| float64     |  0 |
| complex64     |  0+0i |
| complex128     | 0+0i |
| bool     |  false |
| string     |  "" |
| interface     |  nil |
| slice     |  nil |
| channel     |  nil |
| map     |  nil |
| pointer     |  nil |
| function     |  nil |
| struct     |  براساس مقدار پیش فرض فیلدها|
